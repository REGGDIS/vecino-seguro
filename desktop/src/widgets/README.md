# Widgets reutilizables PySide6

Esta carpeta contendrá componentes reutilizables de la aplicación desktop de VecinoSeguro.

Aquí podrán ubicarse widgets PySide6 compartidos entre pantallas, como botones con estilos comunes, tarjetas de resumen, badges de estado, indicadores de prioridad o componentes de formulario. La carpeta queda preparada para crecer sin mezclar componentes reutilizables dentro de vistas específicas.
