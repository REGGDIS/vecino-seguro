# Estilos QSS desktop

Esta carpeta contendrá los estilos QSS de la aplicación desktop de VecinoSeguro.

Los archivos QSS permiten centralizar colores, espaciados, tipografías y estados visuales de los widgets PySide6. La intención es mantener la identidad visual separada de las vistas para que las pantallas no acumulen reglas de presentación difíciles de mantener.

`main.qss` será el punto de partida para los estilos globales de la aplicación.
